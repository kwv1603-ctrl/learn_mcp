## **6. 财务快照与利润纯度**

### **6.1 主人盈余 (Owner Earnings) — [计算年度]**

**⚠️ 数据交叉与保真协议 (MANDATORY)**：
1. **防合并陷阱**：对于 A 股标的，若通用 API 将《无形资产/特许经营权》投资与《固定资产》合并为巨额 CapEx，导致自由现金流失真，**必须通过 Web Search** 强制寻找该年度年报附注的原始拆分数据。
2. **零捏造原则**：绝对禁止按比例倒推或编造拆分数据。如果穷尽搜索仍未获取到附注拆分原件，**必须在此处显式写明：“未获取到结构化拆分明细，本数值基于合并项计算 / 该项缺失”。**
3. **截面一致性**：计算涉及的所有数据变量（利润、折旧、开支）必须统一来自同一会计基础年（例如必须全为 2024 年度原值，或全为 2025 审计值），**严禁跨年度拼凑公式数据**。
4. **数据来源优先级**：`年报原件审计数据 > 官方公告/交易所披露 > API 抓取数据 (yfinance/akshare) > Web 搜索结果 > AI 推理估算`。高优先级来源与低优先级冲突时，以高优先级为准。
5. **口径一致性 (CRITICAL)**：主人盈余公式中的净利润、D&A、CapEx **必须统一使用同一口径**——要么全部为"合并报表"数据，要么全部为"归母"数据。若使用归母净利润，则 D&A 和 CapEx 也须按少数股东权益比例调整；若使用合并口径，净利润须用"合并净利润"而非"归母净利润"。计算公式中**必须显式标注所用口径**（如：$72.97亿 [归母口径] 或 $80.12亿 [合并口径]）。

### **6.2 内在价值快板 (Intrinsic Value)**

> ⚠️ **利率调整强制条件 (MANDATORY)**：若标的所在市场的无风险利率与 Graham 原始假设 (4.4%) 偏差超过 ±1.5%，**必须使用利率调整版公式**。原版 Graham 仅在美元资产且当前利率接近 4.4% 时可单独使用。A 股标的（中国10年国债收益率通常 1.5%~3.5%）**通常必须使用调整版**。

```text
EPS ([年度]):         $[金额]
g (盈利增长率):      [百分比]% (保守/防守型假设)
Y (当前无风险利率):   [百分比]% ([来源：如中国10年国债/美国10年国债])

基础 Graham V = EPS × (8.5 + 2 × g) = $[结果]
利率调整 Graham V_adj = V × (4.4 / Y) = $[调整后结果]  ← ⚠️ 偏差超 +/-1.5% 时必须使用此值
(可结合充沛自由现金或业务护城河进行中轴说明)

Graham 安全边际: ([内在价值中轴] - [现价]) / [现价] = [百分比]%，评价偏离程度。
```

### **6.4 全收益投影模型 (Total Return Framework)**

> **核心逻辑**: 将生意的本质回报 (ROE)、投资的性价比 (隐含收益率) 与股东的实际获得 (股息+增长) 进行联动。

**⚠️ 收益率自洽性协议 (MANDATORY)**:
1. **Yield 倒挂检查**: 若 **股息率 (Dividend Yield) > 隐含收益率 (OE Yield)**，必须在逻辑说明中明确解释现金来源（如：动用往年未分配利润、专项债融资分红、或一次性资产处置）。若无特殊原因，通常暗示主人盈余计算中“维护性开支”设定过高，必须重新核实。
2. **前后一致性**: 确保此处使用的股息率口径（Trailing 或 Forward）与第 1 章及第 9 章保持一致，严禁在同一报告中混用不同年份的收益率因子进行投影。
3. **ROE 锚定校验**: 若 OE Yield 显著低于 ROE，股价必然处于 PB > 1 的溢价状态。需评估留存盈余 (OE Yield - 股息率) 是否能支撑起对应的 BVPS 增速。
